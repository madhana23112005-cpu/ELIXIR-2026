## 🎯 About Elixir'26

**Elixir'26** is the official annual technical symposium of the **EEE Department, Government College of Engineering, Erode (GCE Erode)** — affiliated with Anna University. This repository contains the official symposium website, built to showcase events, manage registrations, and represent the spirit of the fest.
